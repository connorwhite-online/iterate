`iterate-ui` is the umbrella package for iterate's framework adapters — it re-exports `iterate-ui-next` and `iterate-ui-vite` so you can install a single dependency regardless of which framework you use.

```bash
npm i iterate-ui
npx iterate init
```

For full documentation and usage instructions, see the [GitHub README](https://github.com/connorwhite-online/iterate) or the [docs at iterate-ui.com](https://iterate-ui.com).
